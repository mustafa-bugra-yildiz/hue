# Hue Design

If there is one thing I have liked between different programming languages
(and I have studied a lot of languages) it is functions.

## Functions

So functions in Hue are the first and foremost, the most important thing.
Basic syntax is as follows:

```
a -> b -> a + b
```

This is hard to explain, but let me try.

- Every function takes 1 argument and evaluates to an expression.
  This may be another function.

- In this case, `a ->` begins the function, `a` is the argument.
  Which returns another function `b ->`.

- Types are inferred. I don't care if you want explicit types, I have no time.
